На входе строка с выражением, написанным римскими цифрами, на выходе - результат, выражения также в римском виде.
Например:
Ввод: (III+IX)*II
Вывод: XXIV
Допускается использование скобок (и вложенные скобки), знака сложения, вычитания и умножения: () + - *
Результат не превышает 3999.

Symbol       Value
I             1
V             5
X             10
L             50
C             100
D             500
M             1000
